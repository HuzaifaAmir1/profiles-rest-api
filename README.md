## 🚀 Setup Instructions

1. Clone the repo:
   git clone https://github.com/HuzaifaAmir1/profiles-rest-api.git

2. Create virtual environment:
   python -m venv venv

3. Activate:
   venv\Scripts\activate (Windows)

4. Install dependencies:
   pip install -r requirements.txt

5. Run server:
   python manage.py runserver
